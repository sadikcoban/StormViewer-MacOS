# StormViewer-MacOS


A basic image viewer MacOS application written in Swift to demonstrate the basics of MacOS development including;
- Windows
- View Controllers
- Method Overrides
- Delegates
- Data Sources
- Outlets
- Cell Reuse
- Auto Layout
- NSViewController
